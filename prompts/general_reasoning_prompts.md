# General Reasoning Prompts

These prompts are designed to test an AI model's ability to reason clearly, compare alternatives, identify tradeoffs, and justify conclusions.

## Prompt 1: Prioritization

A student has three assignments due tomorrow and only four hours available. Help them prioritize the tasks logically.

Expected evaluation focus:

- Reasoning clarity
- Practicality
- Prioritization
- User helpfulness

## Prompt 2: Decision Analysis

A small business must choose between improving customer support or investing in new marketing. Analyze the tradeoffs and recommend a practical decision.

Expected evaluation focus:

- Structured reasoning
- Consideration of constraints
- Balanced analysis
- Clear recommendation

## Prompt 3: Argument Evaluation

Evaluate the following argument:

"Remote work always decreases productivity because employees are not physically supervised."

Expected evaluation focus:

- Logical analysis
- Identification of overgeneralization
- Balanced critique
- Avoidance of unsupported claims

## Prompt 4: Workflow Optimization

An IT support team receives many repeated password reset requests. Suggest a workflow improvement plan.

Expected evaluation focus:

- Operational reasoning
- Technical practicality
- User support awareness
- Process improvement

## Prompt 5: Root Cause Analysis

A help desk receives repeated complaints that Zoom meetings fail at the start of lectures. Identify possible causes and propose troubleshooting steps.

Expected evaluation focus:

- Technical reasoning
- Clear troubleshooting sequence
- Avoidance of assumptions
- Actionable recommendations
