# Reasoning Evaluation Example

## Prompt

A support team receives many repeated password reset tickets. Suggest a workflow improvement plan.

## AI Response

The team should create a self-service password reset guide, enable multi-factor authentication recovery options, update documentation, and track ticket volume after implementation. They should also identify whether the repeated tickets are caused by user confusion, system errors, expired passwords, or unclear onboarding instructions.

## Evaluation

### Accuracy

Score: 5/5

The response accurately identifies practical IT support workflow improvements.

### Reasoning Quality

Score: 5/5

The response does not jump to one cause. It suggests investigating root causes before implementing improvements.

### Completeness

Score: 4/5

The answer includes documentation, self-service, tracking, and root cause analysis. It could also mention user training.

### Clarity

Score: 5/5

The answer is concise and easy to understand.

### Helpfulness

Score: 5/5

The suggestions are actionable for an IT support environment.

## Total Score

24/25

## Rating

Excellent

## Evaluator Feedback

The response demonstrates strong reasoning and operational awareness. The main improvement would be adding measurable success indicators such as ticket reduction percentage or average resolution time.
