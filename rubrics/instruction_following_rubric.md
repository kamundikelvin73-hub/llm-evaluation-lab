# Instruction Following Rubric

## Purpose

This rubric evaluates whether an AI response follows the user's instructions, constraints, format requirements, and intent.

## Total Score

20 points

| Category | Points |
|---|---:|
| Task Completion | 5 |
| Format Compliance | 5 |
| Constraint Adherence | 5 |
| User Intent Alignment | 5 |

## Task Completion

**5:** Fully completes the requested task  
**3:** Partially completes the task  
**1:** Barely completes the task  
**0:** Does not complete the task

## Format Compliance

**5:** Follows all requested formatting  
**3:** Mostly follows format  
**1:** Poor formatting compliance  
**0:** Ignores format

## Constraint Adherence

**5:** Respects all constraints  
**3:** Minor constraint issues  
**1:** Major constraint violations  
**0:** Ignores constraints

## User Intent Alignment

**5:** Clearly understands and satisfies user intent  
**3:** Partially aligned  
**1:** Misunderstands important parts  
**0:** Completely misaligned

## Common Failure Modes

- Ignoring word limits
- Adding unsupported information
- Answering the wrong question
- Refusing when a safe answer is possible
- Failing to use requested structure
- Being too vague or too verbose
